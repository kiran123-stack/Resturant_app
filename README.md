
# Restaurant App

Restaurant App is a comprehensive, user-friendly application designed to modernize the dining experience. It empowers customers to explore menus, place food orders, and make table reservations with ease. For restaurant management, it provides an intuitive interface to handle orders and customer interactions efficiently.

## üöÄ Features

### For Customers
- **Interactive Digital Menu**: Browse dishes with rich visuals and detailed descriptions.
- **Table Reservations**: Easy-to-use booking system to reserve tables in advance.
- **Online Ordering**: Seamless process for placing food orders.
- **Location Services**: Interactive maps to locate the restaurant easily (powered by Leaflet).
- **Responsive Experience**: A fully responsive design that works perfectly on mobile, tablet, and desktop.

### UI/UX
- **Modern Interface**: Clean and aesthetic design using Tailwind CSS.
- **Smooth Animations**: Enhanced user experience with GSAP and Tailwind Animate.
- **Fast Navigation**: Instant page transitions using React Router.

## üì¶ Installation & Setup

Follow these steps to get the project running on your local machine.

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/Resturant.git
    cd Resturant
    ```

2.  **Navigate to the frontend**
    ```bash
    cd frontend
    ```

3.  **Install dependencies**
    ```bash
    npm install
    ```

4.  **Start the development server**
    ```bash
    npm run dev
    ```

5.  **Build for production**
    ```bash
    npm run build
    ```
